This is a facebook style site, which performs the following functionalities

1) Register user
2) Login as user
3) Post  a comment
4) Reply to a comment
5) All Posts and comments arranged in chronological order

Technologies
-------------
1) ASP.NET
2) AngularJS
3) JQuery
